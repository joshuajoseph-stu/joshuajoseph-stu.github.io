# My Portfolio

An attempt at making a portfolio using React that is actually interesting without being 'too much'. Does it count as vibecoded if im autistically looking at every single line being changed after copy pasting stuff from chatgpt and claude? ig it should lol

## Current features
1. Terminal style portfolio 
2. dir structures that can be navigated with cd
3. files that can be read using cat, and so on 
4. some classic + custom commands

## Planned features
1. An AI chatbot (client side) that helps users who have no idea what a terminal is
2. More files, commands
3. Oh and actually adding content to files xd
